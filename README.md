# Hospital & Pharmacy Power BI Dashboard

Assignment Files
- Power BI Dashboard* (.pbix) - Interactive dashboard with complete analysis
- Written Insights (.docx) - Executive summary (573 words)

 Project Overview
Analysis of 300 patient visits across 6 counties in Kenya throughout 2024, examining disease patterns, pharmacy performance, and healthcare resource utilization.

 Key Findings
- Total Pharmacy Revenue:** KES 275,407
- Average Hospital Stay:** 1.41 days
- Most Common Diseases:** Flu (56 cases), Typhoid (54 cases), Diabetes (53 cases)
- Regional Alert: Kisumu shows Hypertension crisis (30% of visits)
- Age Impact: Seniors (65+) account for 37% of pharmacy revenue
